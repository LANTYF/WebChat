# 即时聊天项目
##前端使用vue+vant 后端使用node(express)+mysql
## 项目完成度
+ 1对1即时聊天
+ 搜索列表添加好友
+ 好友申请是否通过
+ 好友状态即时更新
+ 消息界面显示每个好友发送的最新消息
+ 用户上线后消息界面显示离线时收到的消息
